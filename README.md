### 🗂 プロジェクト紹介

BTCJAPANは、ビットコインプレッジチェーンBouncebitをベースに構築されたAIプラットフォームで、チェーンはEVM等をサポートしているため、Googleやイーサリアムのウォレット（MetaMask、OKXなど）へのアクセスをサポートしています。 主要なユーザーはアジアに集中しています。

ここでは、対応するタスクを完了して SBT をクレイムし、ランダムな AI デジタル ガールフレンドを取得できます。 日々の交流や仲間づくりを通じてポイントを貯めることで、彼女は自意識を成長させ、AI駆動のアイドル少女へと成長していきます。
